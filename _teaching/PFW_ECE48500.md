An introduction to embedded real-time operating systems, with an emphasis on embedded system software development, tasks, inter-task communications and synchronization as well as network software.

<h2>Course Coverage</h2>
The general course schedule is listed below:

<ul>
<li> Introduction </li> 
<li> Review of C and Embedded C </li> 
<li> Coding Style and Intellectual Property </li> 
<li> Launchpad GPIO and LCD </li> 
<li> Launchpad Hardware Configurations </li> 
<li> Launchpad Advanced Features and Access Level </li> 
<li> Operating System Concepts </li> 
<li> FreeRTOS Realization Basic and Task </li> 
<li> FreeRTOS Inter-process Communication </li> 
<li> File System </li> 
<li> Real-Time Scheduling Algorithm </li> 
<li> Conclusion </li> 

</ul>

This course uses FreeRTOS with Texas Instrument TM4C123 Tiva Board.
