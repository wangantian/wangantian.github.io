An introduction to digital system design and hardware engineering, with an emphasis on practical design techniques and circuit implementation.  


<h2>Course Coverage</h2>
This course cover typical content as the first course for the digital logic course. The general course schedule is listed below: 

<ul>
	<li>	Introduction	</li> 
	<li>	Number Representation and Conversions	</li> 
	<li>	Digital Signal Conversion and Transmission	</li> 
	<li>	Logic Gates and Reconfigurable Logic Realization	</li> 
	<li>	Boolean Algebra Computation and Simplification	</li> 
	<li>	Combinational Logic Timing Analysis, XOR/XNOR, and Multiplexers	</li> 
	<li>	Binary Arithmetic Combinational Logic Design	</li> 
	<li>	Encoder and Decoder Design for Combinational Logic	</li> 
	<li>	Latches and Flip-Flops	</li> 
	<li>	Finite State Machine: Design, Analysis, and State Reduction	</li> 
	<li>	Timing Analysis in Sequential Logic, and Counter, Shift Registers	</li> 
	<li>	Multivibrators and the 555 Timers	</li> 
	<li>	An Intense Introduction of Circuit Aspect of Digital Logic	</li> 
	<li>	Memory Technology 	</li> 
	<li>	Conclusion	</li> 
</ul>
