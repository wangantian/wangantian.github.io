<p>This page provides an evolving summary of resources and notes for using the AUP-ZU3 FPGA board. The AUP-ZU3 is a newer Zynq-based FPGA development board, available <a href="https://www.realdigital.org/hardware/aup-zu3">here</a>. It is currently available for purchase only by academic institutions. Consider purchasing the complete kit, especially if you do not already have the required cables, SD card, and power adapter.</p>

<p>The official documentation is available <a href="https://xilinx.github.io/AUP-ZU3/">here</a>.</p>
